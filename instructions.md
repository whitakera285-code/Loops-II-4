Write a function called `findCumulativeSum` that takes one parameter, a number, and returns the sum of all positive integers from 0 up to and including that number (you can assume you'll never be given a negative number). 

**Hint**: You might need to have a separate tracking variable (perhaps named `sum`?) outside of your loop that's keeping track of the running total.

For Example:

```
> findCumulativeSum(5);    // Finds sum of 0 + 1 + 2 + 3 + 4 + 5
```

```
15
```
